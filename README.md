# Modélisation géométrique

Divers algorithmes permettant de modéliser des formes et de les décomposer en facettes triangulaires sur Unity.

Permet aussi de lire des fichiers de maillage au format `.off`.

Formes déjà faites :
* Plan
* Cylindre
* Sphère
* Cône
